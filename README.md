# Evolved Snake

## Vercel Link

https://snake-game-one-kappa-51vercelapp.vercel.app

## Design Concept

起初先根據三段式開發設計出原始版的貪吃蛇，後續在玩法設計上以原始版為基礎加入難度與原始版做出差異。中等裡起初為了與原始版做出差異調整了速度與水果數量，因為是初步測試所以差異度不大。而在困難中，嘗試加入的新道具的機制設計，首先是兩塊的石頭障礙，讓原本只有25方格的空間更加限縮，後續為了讓玩家在有限的空間中不會因為身體的延伸過多限制到角色的移動，而加入了新道具花朵讓玩家能去減少角色的身體長度，考量到不止困難中會因為空間限制而影響遊玩，所以將道具下放到前面簡單與中等裡。為了加入更多的差異與挑戰性，最後在困難中加入藍莓代表腐爛水果，會去扣除玩家的分數。

## How My Version Is Different

與傳統貪吃蛇擁有廣大棋盤、目標只是讓蛇不斷變長不同，進化版中加入了具有不同效果的道具：一般水果能增加分數，花朵（🌸）會減少蛇的長度，腐爛水果（🫐）則會扣分。難度不僅影響速度，還會改變水果數量、加入障礙物、以及新道具的機制。遊戲也具備暫停/繼續功能與鍵盤控制，適合快速遊玩。

## How To Play

1. 在主頁面選擇難度（Easy / Medium / Hard）
2. 按下方向鍵或 WASD 開始遊戲
3. 控制蛇吃掉水果（🍎🍊🍋🍇🍉）來增加分數
4. 吃掉縮小花朵（🌸）可以讓蛇變短
5. 在 Hard 模式中，避開腐爛水果（🫐），否則會扣分
6. 避免撞牆、障礙物（🪨）或蛇自己的身體
7. 按下 **空白鍵** 或 **Escape** 暫停/繼續遊戲
