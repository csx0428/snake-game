# Evolved Snake

## Vercel Link

https://snake-game-one-kappa-51vercelapp.vercel.app

## Design Concept

Evolved Snake 將經典街機遊戲貪吃蛇重新想像為一款在 5×5 緊湊棋盤上進行的 tactical 解謎體驗。核心概念是將貪吃蛇從無限成長的空間管理遊戲，轉變為每一步都至關重要的策略決策挑戰。

小巧的棋盤迫使玩家必須提前思考數步，因為蛇身會迅速填滿有限的空間。為了增加深度，遊戲引入了特殊水果：縮小花朵（🌸）能讓蛇縮短，但同時也損失了累積的長度；腐爛水果（🫐）則會在困難模式中扣減分數。這些元素創造了風險與報酬之間的取捨，讓每一局遊戲都充滿張力。

三種難度等級滿足了不同玩家的需求。簡易模式提供輕鬆的節奏，只會出現單一水果，縮小門檻也較寬鬆。中等模式同時出現多種水果，增加了認知負荷。困難模式則加入了障礙物和腐爛水果機制，考驗玩家的路徑規劃能力。

視覺設計採用暗色 cyberpunk 風格，搭配霓虹綠 accents、發光的格子邊框和漂浮動畫，打造出 polished 且現代的遊戲氛圍，同時不干擾 gameplay。

## How My Version Is Different

與傳統貪吃蛇擁有廣大棋盤、目標只是讓蛇不斷變長不同，Evolved Snake 將遊戲壓縮到 5×5 的密集空間中，每一寸空間都至關重要。本版本引入了多種具有不同效果的水果：一般水果能增加分數，縮小花朵（🌸）會減少蛇的長度，腐爛水果（🫐）則會扣分。難度不僅影響速度，還會改變水果數量、加入障礙物、以及引入全新的機制。遊戲也具備暫停/繼續功能與鍵盤控制，適合快速遊玩。

## How To Play

1. 在主頁面選擇難度（Easy / Medium / Hard）
2. 按下方向鍵或 WASD 開始遊戲
3. 控制蛇吃掉水果（🍎🍊🍋🍇🍉）來增加分數
4. 吃掉縮小花朵（🌸）可以讓蛇變短
5. 在 Hard 模式中，避開腐爛水果（🫐），否則會扣分
6. 避免撞牆、障礙物（🪨）或蛇自己的身體
7. 按下 **空白鍵** 或 **Escape** 暫停/繼續遊戲
